About Me

Saya adalah mahasiswa Program Studi Sistem Informasi di Universitas Islam Negeri Alauddin Makassar (UINAM), Fakultas Sains dan Teknologi, yang mulai menempuh pendidikan pada tahun 2024. Saya memiliki ketertarikan besar pada bidang pengembangan web dan sistem informasi, khususnya dalam merancang dan membangun website yang terstruktur, fungsional, serta mudah digunakan.

Dalam proses perkuliahan dan pengembangan diri, saya terbiasa menggunakan teknologi seperti HTML, CSS, dan JavaScript untuk pengembangan antarmuka website. Selain itu, saya juga memahami konsep dasar pemrograman backend serta pengelolaan basis data menggunakan PostgreSQL dan MySQL.

About Me

Saya adalah mahasiswa Program Studi Sistem Informasi di Universitas Islam Negeri Alauddin Makassar (UINAM), Fakultas Sains dan Teknologi, yang mulai menempuh pendidikan pada tahun 2024. Saya memiliki ketertarikan besar pada bidang pengembangan web dan sistem informasi, khususnya dalam merancang serta membangun website yang terstruktur, fungsional, dan mudah digunakan.

Selama proses perkuliahan dan pengembangan diri, saya terbiasa menggunakan HTML, CSS, dan JavaScript untuk pengembangan antarmuka website. Selain itu, saya juga memahami konsep dasar pemrograman backend serta pengelolaan basis data menggunakan PostgreSQL dan MySQL.

Saya dikenal sebagai pribadi yang bertanggung jawab, disiplin, serta mampu bekerja baik secara mandiri maupun dalam tim. Saya memiliki semangat belajar yang tinggi terhadap teknologi baru dan berkomitmen untuk terus meningkatkan kemampuan di bidang teknologi informasi.

Melalui website portofolio ini, saya ingin menampilkan profil, kemampuan, serta proyek yang pernah saya kerjakan sebagai langkah awal menuju dunia profesional.
